Aplicando Estilos


1. Aplicando estilo desde la misma etiqueta Html

            
                <p> style="font-size: 20px;">Mi Texto <p>

2. Aplicado estilo como etiqueta style desde la estructura html

              
                <style>
                    p {
                        font-size: 20px;
                       }
                <style>

3. Como archivo externo

            
                <link rel="stylesheet" href="misestilos.css">
                Archivo externo
                p {
                     font-size: 20px;
                }
