Referencia a Estilos


1. Por Nombre

            
                span {
                    font-size: 20px;
                      }
            
            Se aplica los cambios a todas las etiquetas 'span'

2. Múltiples selectores

              
                p, span {
                            font-size: 20px;
                        }
            
            Se aplica los cambios a todas las etiquetas 'span' y 'p'
            Se referencia a varias etiquetas, todas estas separadas por una coma

3. Selector descendiente

            
                main p {
                    font-size: 20px;
                   }
            
            Se aplica los cambios a todas las etiquetas 'p' que esten dentro de la etiqueta main
            La etiqueta tiene que estar dentro de la main, ya siendo hijo o nieta de esta

4. Selector hijo directo

            
                section > p {
					font-size: 20px;
				}
            
            Se aplica los cambios a todas las etiquetas 'p' que esten dentro de la etiqueta 'section'
            La etiqueta tiene que estar dentro de section directamente, tiene que ser hijo y no nieto

5. Selector de predecesot

            
                h1 + p {
                    font-size: 20px;
                    }
        
            
            Se aplica los estilos a la etiqueta p que se encuentre después de la etiqueta h1
            Tienen que sean Hermanos y este inmediatamente después de h1. 
            Caso existe consecuente otra etiqueta p o esta este en medio de otra etiqueta,
            no se podrá efectuar el estilo.
